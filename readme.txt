Customize expire time for user and admin logged cookies

== Description ==

Using this plugin cookies expire time can be extended based on custom value provided in settings page

== Installation ==

1. Upload `Wpp_cookie_expire` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. click on `General` from left side panel to modify cookie expire settings

== Frequently Asked Questions ==

= Where can i found the settings for this plugin =

You can found it in wordpress general settings section
